# Climbing Plan Web App

## Overview
This web app is designed to help users track and plan their climbing activities, providing features to log climbs, set goals, and monitor progress.

## Features
- **User Registration and Login**: Users can create an account and log in to access their climbing plans.
- **Climbing Log**: Record every climb with details such as date, location, climb type, and difficulty.
- **Goals Tracker**: Set personal climbing goals and track progress.
- **Statistics Dashboard**: Visual representation of climbing activities and achievements.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **Database**: MongoDB

## Installation
To run the web app locally:
1. Clone the repository: `git clone https://github.com/font1/playground.git`
2. Navigate to the project directory: `cd playground`
3. Install dependencies: `npm install`
4. Start the server: `npm start`

## Usage
Once the app is running, navigate to `http://localhost:3000` to access the climbing plan web app.
